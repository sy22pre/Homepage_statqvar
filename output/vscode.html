VScodeについて

Visual Studio Code(VScodeと呼ばれることが多いです)とは，いわゆるエディターと呼ばれるものです． エディターとは，何かを編集するときに用いるソフトのことです． つまりプログラミングを書いたり，文字をテキストに書いたりするときに使います．

VScodeは，編集できるテキストファイルが多く拡張機能が多いのが特徴です． VScodeは使っている人が多いため知りたい情報をネットで検索するとすぐに調べることができます． VScodeは，自分が使いやすいようにカスタマイズできるので調べてみてカスタマイズしていってください． では早速導入の手順を説明していきます．

VScodeのインストール

Windows11では，MicroSoft StoreでインストールできるようになっているのでWindows11の方はそこからインストールしてください． (作者はWindows10のため確認できてないです．) その他のWindowsの方は，こちらで検索してください． 次のような画面が出てくると思いますので，「Download for Windows」をおして次の画面のインストールしてください．

インストールされた「VSCodeUserSetup.exe」起動し契約書を同意し右下の「次へ」を押してください． インストール先の指定は，何もいじらずそのまま「次へ」を押してください． スタートメニューフォルダーの指定は各自の判断でお願いします． 次の画面はデフォルトのままでも大丈夫です． (個人的にはアイコンは追加しても良いのかなと思います)

これで最後の画面になると思うのでそのまま右下の「インストール」を押してインストールを開始してください．

ショートカットキーについて

ここで，ショートカットキーについても説明していきたいと思います． ショートカットキーとは，キーボードを使いパソコンの操作を行うための機能です． 使えるようになると便利なので覚えていましょう！

  • Ctrl + s : 保存
  • Ctrl + x : 切り取り
  • Ctrl + c : コピー
  • Ctrl + v : 貼り付け
  • Ctrl + z : 一つ前に戻る
  • Ctrl + o : ファイルを開く
  • Ctrl + n : 新しいファイルを開く

これは，Windows上ならだいたい使えるショートカットキーです． 以下は主にVSCode上で使うショートカットキーです．

  • Ctrl + d : 単語選択
  • Shift + → or ← : 文字の選択
  • Ctrl + [ or ]: インデントを削除or挿入
  • Ctrl + / : コメントアウト

インデントとは，段落単位で文字の位置をそろえる機能です． プログラムの見やすさやpyhonのfor文などを管理するために重宝するのでおいてください． 主に使うのはこんな感じですが，ショートカットキーはまだまだあったりVSCode上でのみ使えるショートカットキーを自分で作れたりするので活用してください．
